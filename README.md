# NinjaTools
Development tools for development ninjas.

## Content

### osc
A command line tool for fetching screenshots for LAN connected oscilloscopes.

### osc_usb
A USB based legacy version of osc. Not recommended for use due to requiring often poor vendor USB drivers for the scope.

### allegro_skill

Utility skill scripts for OrCAD/Allegro PCB Designer + instructions for how to auto load them.

### allegro_hotkeys

Hotkeys/settings (env file) for OrCAD/Allegro PCB Designer + configurations for a Keychron K0 Max macropad to use them.

### allegro_jlcpcb
A very basic script for renaming OrCAD/Allegro PCB Designer artwork files to the naming format required by JLCPCB.
